# Caso 02 — Inicio de sesión

- Precondición: Cuenta existente (usar credenciales de prueba o creadas en caso 01)
- Pasos:
  1. Abrir `/`
  2. Seleccionar `Iniciar sesión`
  3. Ingresar correo y contraseña
  4. Enviar formulario
- Resultado esperado: Redirección a `/admin` y presencia del nombre en el header.
- Evidencia: captura de pantalla y respuesta del endpoint `/api/auth/login`.
